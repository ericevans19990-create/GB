Dev Info — projet

Endpoints

  • Frontend (Vite): http://localhost:5173/ (ou port affiché par Vite)
  • API stub: http://localhost:3001
  • Health: /api/health → ex: http://localhost:3001/api/health
  • Form submit: POST /api/form

Commandes utiles

Tester l'API (curl) :

curl -sS http://127.0.0.1:3001/api/health

Envoyer une soumission de test :

curl -sS -X POST http://127.0.0.1:3001/api/form -H "Content-Type: application/json" -d '{"formId":"contact","data":{"name":"Test","email":"t@e.com"}}'

Images locales

Les images sont dans /public/images/. Exemples :

  • /images/logo.svg
  • /images/hero.jpg
  • /images/services.jpg
  • /images/blog-thumb.jpg
  • /images/team-1.jpg
  • /images/team-2.jpg
  • /images/team-3.jpg

Pour remplacer : déposer vos fichiers dans public/images/ (même nom), ou modifier les composants pour pointer vers un autre fichier.
